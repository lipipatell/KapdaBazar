KapdaBazar is a PHP & MySQL based e-commerce website.

## Features
- Product gallery
- Search & filter
- Add to cart
- Wishlist
- Session-based cart
- MySQL database

## Tech Stack
- PHP
- MySQL
- HTML, CSS
- Bootstrap
- XAMPP

## Database Setup
1. Open phpMyAdmin
2. Create database `gallery_db`
3. Import file: `database/gallery_db.sql`

## Note
Database credentials are stored in `config.php` (not included for security).
